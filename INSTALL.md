# Fish Again Finder Install

Fish Again Finder is a Firefox extension with simple launchers for Mac and Windows.

## Requirements

- Firefox
- Node.js/npm

## Mac

1. Download and unzip `fish-again-finder-mac.zip`.
2. Double-click `launch-fish-again-mac.command`.
3. Use the Firefox window that opens.

If macOS blocks the launcher, right-click it and choose `Open`.

## Windows

1. Download and unzip `fish-again-finder-windows.zip`.
2. Double-click `launch-fish-again-windows.bat`.
3. Use the Firefox window that opens.

## Usage

- Open the page with the `Fish Again` button in the launched Firefox window.
- Use the floating panel to Start/Pause.
- Drag the panel header to move it.
- Drag the bottom-right corner to resize it.
- Change `Every ___ sec` to control the click period.

The extension clicks the newest on-screen `Fish Again` button and scrolls to the bottom after each click.

## Note

Firefox treats this as a temporary local extension. For a normal permanent add-on install, the extension has to be signed through Mozilla Add-ons.
